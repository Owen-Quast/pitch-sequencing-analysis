# MLB Pitch Sequencing Analysis (2023)

This project analyzes how pitch sequencing affects whiff rates using MLB Statcast data.

## Project Goals
Investigate how the pitch thrown previously affects the effectiveness of the next pitch.

## Analysis Includes
- Fastball effectiveness based on the previous pitch
- Whiff rates across pitch-to-pitch sequences
- "Setup Value" metric (whiff rate above baseline)
- Network visualization of pitch sequencing strategy

## Data Source
Statcast via the `baseballr` R package

## Tools Used
- R
- tidyverse
- baseballr
- ggplot2
- igraph / ggraph
